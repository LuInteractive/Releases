# 1.30.0.0
This release brings the new August application

## New application
* The application **Grööve** launched! Let yourself be carried away by the rhythm of the music and learn choreographies that groove! Two modes are available: a practice mode that breaks down the choreography into movement sequences, and a free dance mode that allows you to simply learn it as you go. Enjoy the show!
* A new version of LuUpdate is also included in this release. The new LuUpdate features a higher level of stability and reliability. A brand new interface brings a high quality UX and brand new analytics are also in place
* A new localization system is now used accross the entire application, which simplifies a lot the integration workflow for the development team
* A couple general bug fixes and improvements were also included
